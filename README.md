# AI Career Guidance App

An AI-powered Career Guidance Application built using Flutter and Firebase to help students discover suitable career paths, assess their skills, and receive personalized guidance through Artificial Intelligence.

## Overview

Choosing the right career path can be challenging for students due to the large number of available options and rapidly changing industry requirements. This application aims to simplify career decision-making by providing personalized career recommendations, AI-powered guidance, skill assessments, and learning suggestions.

## Installation

1. Clone the repository

```bash
git clone https://github.com/shahrprati14020706/AI-career-guidance-app.git
```

2. Navigate to the project

```bash
cd AI-career-guidance-app
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
